# Asterfold 2.1.0

Готовое расширение для новой вкладки Chrome. Сайты раскладываются по понятным блокам и остаются на вашем компьютере.

## Что изменилось в 2.1.0

- Обычный клик по закладке открывает сайт **в этой же вкладке**. Новые закладки и закладки, созданные в прошлых версиях, используют это поведение автоматически. В редакторе по-прежнему можно явно выбрать новое окно или инкогнито.
- Интерфейс теперь поддерживает **русский, қазақша, English, Español, Deutsch, Français, Italiano, Português, Polski, Українська, Türkçe и Nederlands**.
- Режим языка **«Авто»** берёт язык интерфейса Chrome. Например, заголовок вкладки становится `New Tab`, `Новая вкладка`, `Жаңа қойынды` или `Nueva pestaña`.
- Настройки и Quick Save используют тот же выбранный язык. Для редких строк европейских языков есть понятный английский fallback — интерфейс не возвращается к русскому.
- Локальная схема IndexedDB обновляется до v5 без удаления страниц, блоков, закладок или обоев.

## Что осталось из 2.0.3

- Контекстное меню открывается поверх блоков, автоматически отступает от краёв и остаётся компактным.
- Стекло, обои и прозрачность включены по умолчанию. Режим для слабых ПК и отдельные анимации включаются только вручную.

## Самый простой способ установить

1. На странице этого релиза скачайте **`Asterfold-Chrome.zip`**.
2. Распакуйте ZIP в любую постоянную папку, например «Документы/Asterfold».
3. В Chrome откройте `chrome://extensions`.
4. Включите **Режим разработчика** справа вверху.
5. Нажмите **Загрузить распакованное расширение**.
6. Выберите распакованную папку, где видно `manifest.json`.
7. Откройте новую вкладку Chrome.

Готово: маленький знак Asterfold находится внизу слева. Через него можно создавать блоки, искать ссылки и открывать настройки.

## Что умеет версия 2.1.0

- страницы, блоки и закладки с перетаскиванием;
- Quick Save для сохранения текущего сайта;
- поиск, корзина и Privacy Mode;
- 12 языков и автоопределение языка Chrome;
- светлая/тёмная тема, обои, эффект стекла и ручной режим для слабых ПК;
- резервные копии и импорт закладок;
- хранение данных локально в браузере — регистрация не нужна.

## Файлы релиза

- **`Asterfold-Chrome.zip`** — единственный файл для установки в Chrome.
- **`extension-source.zip`** — исходный код только для разработчиков.
- **`checksums.txt`** — контрольные суммы файлов.

Не используйте автоматические GitHub-файлы **Source code (zip)** и **Source code (tar.gz)** для установки — это архивы исходников, а не Chrome-расширение.

Если обновляете расширение, распаковывайте новую версию в ту же папку и в `chrome://extensions` нажимайте кнопку обновления у Asterfold.
