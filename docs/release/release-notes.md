# Asterfold 2.0.3

Готовое расширение для новой вкладки Chrome. Сайты раскладываются по понятным блокам и остаются на вашем компьютере.

## Что изменилось

В `2.0.3` исправлены контекстные меню и управление производительностью. Это обновление не меняет ваши закладки, страницы или настройки.

- Контекстное меню теперь всегда открывается поверх блоков и не прячется за соседними карточками.
- У правого, нижнего и остальных краёв окна меню автоматически смещается так, чтобы все его пункты оставались видны.
- Меню стало компактнее, но сохраняет все действия.
- Эффект стекла, обои и прозрачность снова включены по умолчанию.
- Режим для слабых ПК теперь включается только вручную в «Настройки → Внешний вид».
- Анимации закладок, меню и перестановки блоков можно отключать отдельно.

В `2.0.2` Asterfold стал заметно легче для старых компьютеров и встроенной графики. Это обновление не меняет ваши закладки, страницы или настройки.

- На слабых компьютерах автоматически включается простой режим отображения.
- Эффект стекла, размытие обоев и тяжёлые переходы отключаются, когда они могут вызывать задержки.
- Если устройство выглядит достаточно мощным, но новая вкладка всё равно пропускает несколько кадров, Asterfold сам переключается на лёгкий режим.
- Во время перетаскивания блоков интерфейс временно использует более дешёвое непрозрачное отображение.
- В обычном режиме дизайн и все возможности остаются прежними.

Версия `2.0.1` уже исправила упаковку Windows/Chrome: главный архив **`Asterfold-Chrome.zip`** содержит `manifest.json` прямо в выбранной папке. Ошибка **“Manifest file is missing or unreadable”** не возникает, если выбрать распакованную папку из этого архива.

## Самый простой способ установить

1. На странице этого релиза скачайте **`Asterfold-Chrome.zip`**.
2. Распакуйте ZIP в любую постоянную папку, например «Документы/Asterfold».
3. В Chrome откройте `chrome://extensions`.
4. Включите **Режим разработчика** справа вверху.
5. Нажмите **Загрузить распакованное расширение**.
6. Выберите распакованную папку, где видно `manifest.json`.
7. Откройте новую вкладку Chrome.

Готово: маленький знак Asterfold находится внизу слева. Через него можно создавать блоки, искать ссылки и открывать настройки.

## Что есть в версии 2.0.3

- страницы, блоки и закладки с перетаскиванием;
- Quick Save для сохранения текущего сайта;
- поиск, корзина и Privacy Mode;
- русский и қазақша языки;
- светлая/тёмная тема, обои и эффект стекла;
- резервные копии и импорт закладок;
- хранение данных локально в браузере — регистрация не нужна.

## Файлы релиза

- **`Asterfold-Chrome.zip`** — единственный файл для установки в Chrome.
- **`extension-source.zip`** — исходный код только для разработчиков.
- **`checksums.txt`** — контрольные суммы файлов.

Не используйте автоматические GitHub-файлы **Source code (zip)** и **Source code (tar.gz)** для установки — это архивы исходников, а не Chrome-расширение.

Если обновляете расширение, распаковывайте новую версию в ту же папку и в `chrome://extensions` нажимайте кнопку обновления у Asterfold.
